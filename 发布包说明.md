# 茶叶监测系统发布包说明

## 1. 发布包内容

本发布包面向最终使用人员，双击即可运行，不需要重新编译项目。

发布包中主要包含以下内容：

- `TeaMonitorFusion.exe`
- Qt 运行库与插件目录
- `SeaSDK.dll`
- `启动茶叶监测系统.vbs`
- 本说明文件

## 2. 推荐启动方式

推荐优先双击以下文件启动软件：

- `启动茶叶监测系统.vbs`

该方式会尽量避免命令行黑色窗口干扰。

如果需要，也可以直接双击：

- `TeaMonitorFusion.exe`

## 3. 适用场景

当前发布包适用于以下场景：

- 软件界面演示
- 使用模拟数据进行流程验证
- 使用如海广电光谱仪并已具备对应运行环境

## 4. 真机使用前的额外条件

如果需要连接真实设备，仅复制发布包通常还不够，还需要目标电脑提前安装对应厂商的驱动或运行环境。

### 4.1 如海广电光谱仪

需要确保目标电脑可以正常加载：

- `SeaSDK.dll`
- 对应设备驱动

本发布包已包含 `SeaSDK.dll`，但若目标电脑没有安装驱动，仍可能无法正常识别设备。

### 4.2 海洋光学光谱仪

海洋光学光谱仪使用的是另一套 SDK。

当前版本项目已经保留了对应适配结构，但如果要在其他电脑上直接连接真机，通常仍需要：

- 安装 OceanDirect 运行环境或 SDK
- 保证目标电脑具备对应驱动与依赖库

### 4.3 工业相机

工业相机当前按 eBUS SDK 方向预留。

如需连接真实工业相机，目标电脑通常还需要：

- Pleora eBUS Runtime / SDK
- 对应网卡或相机驱动环境

## 5. 发布包使用步骤

1. 将整个 `TeaMonitorFusion_release` 文件夹完整复制到目标电脑。
2. 不要只复制 `TeaMonitorFusion.exe`，必须保留同级目录下的所有 `.dll` 和插件文件夹。
3. 推荐先双击 `启动茶叶监测系统.vbs` 运行软件。
4. 若连接真机失败，请先检查目标电脑是否已经安装对应设备驱动和厂商运行环境。

## 6. 目录要求

请保持发布包目录结构完整，不要删除或移动以下内容：

- `platforms`
- `styles`
- `imageformats`
- `iconengines`
- `generic`
- `networkinformation`
- `tls`
- `translations`
- 各类 Qt `.dll`
- `SeaSDK.dll`

否则软件可能无法启动，或启动后缺少图像、主题、网络或设备相关功能。

## 7. 常见问题

### 7.1 双击后无法启动

优先检查：

- 是否误删了 `platforms` 文件夹
- 是否只复制了 `TeaMonitorFusion.exe`
- 是否有安全软件拦截

### 7.2 能打开界面，但无法连接设备

通常不是发布包损坏，而是目标电脑缺少：

- 厂商驱动
- SDK 运行环境
- 正确的设备连接方式

### 7.3 界面图标或样式显示异常

请确认发布包中的 Qt 插件目录和 `.dll` 文件保持完整。
